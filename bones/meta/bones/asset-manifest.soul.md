### Architectural Intent
A YAML manifest generated by `rc-assets.sh` listing all static files found inside rendered HTML, mapping relative paths to their SHA256 checksums to verify asset integrity.

### Directory / File Schema Expectations
The manifest is automatically generated and updated during builds. Any manual comments or fields added to this file will be wiped clean when `rc-assets.sh` executes. Do not modify this manifest manually. If checksums do not match the files in the output directory, assets may be flagged as corrupted or modified.
